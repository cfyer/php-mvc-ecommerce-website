# Simple PHP MVC E-Commerce Website

### Configuration
- Clone the repository
```
git clone https://github.com/cfyer/php-mvc-ecommerce-website.git
```

- Install packages
```
composer install
```

- Build .env file
```
php command.php env
```

- Run server
```
php command.php serve
```

### Features
- Cart
- Authenticate 
  - register
  - login
- Admin Panel
  - Category
  - Product
  - Slider
